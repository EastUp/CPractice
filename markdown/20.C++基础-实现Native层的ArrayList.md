## 2. 编译流程和模板类

模板类：`语法跟模板函数非常相似`

```c++
template <typename T>
class Callback{
public:
    void onError(){

    }

    void onSucceed(T result){
        cout << result << endl;
    }
};

// 模板类继承
// 如果子类不是模板类
class HttpCallback : public Callback<int>{

};

// 子类还是模板类
template <typename T>
class HttpCallback : public Callback<T>{

};
```
