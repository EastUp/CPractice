# 3.如何封装 java 层 sdk 模型

假设我对一张图片做操作  

- 掩膜操作
- 模糊操作
- 又来一个关于 filter2D 的操作

规则考虑周到

- 细节拆分
- 尽量不要改 native 层的代码，可以改 java 层的（`针对上面的需求可以把 filter2D 方法提到 java 层`）
- 思考：时间空间复杂度
